After further discussion with Adam, we do have the capabilities to bind multiple projects to one repo in git.

Turn on bind detection for git in file|options

Create a new “dummy project, bind to git, and push to the repo.

Now create a new project and select the “dummy” projects folder. This will trigger the bind detection
Click yes, then push to the repo. Ini

Click yes, then do a commit- all files will be in a pending add state.

Push to the repo. Now both projects are now in the repo.

To import, you will have to import the dummy project first, then create all other projects in its folder

 

Another solution would be to create a new folder through Windows, then, then use gitbash to initialize it.

Create your new projects in this folder and you will have your bindings
